This tool was built for myself and does not work for any other modes or regions of the game other than Non-Ladder, Softcore, Reign of the Warlock. 
Prices are calculated based on lowest value to highest value version of an item, then returning the median value. User is still required to do some checks to appropriately value items.

It works especially well for Rune values and items that are frequently traded. Items that are not frequently traded do not have recent data and are therefore not as easy to fetch values for.
